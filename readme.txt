Note : The script was written in Python 3.8 so if you haven't installed it run the included installer.
       Display of time may not work if city timezone differs from system timezone

This program allows you to setup a list of cities
and view their weather statistics.

Start by opening window.pyw with Python 3.0 or above.
When you first start this application you're asked to enter
your first city and its country code by global standards (The default is set to "DE").

If you got some entries in your list you can add more or remove selected ones.
To exit just press the save & exit button and your list is saved for the next time.
